# RFLkPC
Robust Fuzzy Local K-Plane Clustering With Mixture Distance of Hinge Loss and L1 Norm

# Overview
This paper proposes a robust fuzzy local k-plane clustering (RFLkPC) method, which incorporates boundedness constraints, hinge loss, and an L1-norm mixed distance. Extensive experiments validate its effectiveness on both synthetic and real-world datasets.
